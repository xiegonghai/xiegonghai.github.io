#tinyhttpd-轻量级httpServer的源码实现

tinyhttpd是一个超轻量型Http Server，使用C语言开发，全部代码只有500、600行，附带一个简单的Client，可以通过阅读这段代码理解一个Http Server的本质。

httpd.c头文件函数声明

每个函数的功能
void accept_request(void );
接受客户端的请求，并对根据请求的类型做相应处理，主要针对get或者post方法解析参数
void bad_request(int);
利用send发送消息来告诉客户端其请求产生了什么问题
void cat(int, FILE );
将file文件的数据写入到socket
void cannot_execute(int);
告诉client这个CGI脚本不能运行
void error_die(const char );
打印错误消息
void execute_cgi(int, const char , const char , const char );
执行一个CGI程序，需要设定一些环境变量
int get_line(int, char *, int);

void headers(int, const char );
返回请求的HTTP头部，200表示请求成功
void not_found(int);
404网页未找到错误,定义一个404页面
void serve_file(int, const char );
发送一个常规文件给客户端
int startup(u_short *);
这个函数主要是在指定端口监听web连接，如果端口是0，动态分配个端口，修改原始端口为实际端口
void unimplemented(int);
如请求方法未实现，告知客户端未实现页面
